# Project Documentation

This directory contains structured project documentation.

## Structure

- `architecture/` — system design and architectural decisions
- `performance/` — load testing methodology, scenarios, and benchmark runs

## Goals

The documentation aims to:

- Provide architectural clarity
- Ensure reproducible performance testing
- Maintain traceability between versions and benchmarks
- Support academic reporting (if applicable)

This documentation is evolving and will be expanded as the project matures.
