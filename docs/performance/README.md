# Performance Testing

This section contains load testing scenarios and archived benchmark runs.

## Structure

- `methodology.md` — testing rules and measurement standards
- `scenarios/` — canonical load testing scenarios
- `runs/` — archived benchmark executions with reports and conclusions

## Goals

- Ensure reproducible performance measurements
- Track performance changes across versions
- Provide transparent comparison between implementations
- Maintain structured and traceable benchmark history

This section will grow as more benchmark runs are added.
